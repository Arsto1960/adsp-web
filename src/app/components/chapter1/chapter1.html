Chapter 1 : Introduction to Digital Signal Processing

Course Overview

This book presents the algorithms with Python examples, and the mathematics behind them, of advanced digital signal processing. It is a course given as an entry level Masters Course for students of Media Technology and Communications and Signal Processing at the Technische Universitaet Ilmenau, in Ilmenau, Germany.

It starts with fundamentals of quantization, continues to more advanced topics like vector quantizers, then to sampling, and to an important mathematical tool for analyzing and synthesizing signals and systems, the z-transform. It then goes on to filter design and Noble identities, allpass filters, warping, minimum phase filters, Hilbert transform, matched- and Wiener filters, and finally predictive filters and their application in predictive coding.

The prerequisites for this course are usually covered by a Bachelor course in Electrical Engineering or related, and are the basics of analog and digital signal processing, and signals and systems.

The Python examples presented here are also in our Github repository: https://github.com/TUIlmenauAMS/ADSP_Tutorials

1.3 Application Example: ITU G.711 Speech Coding

The first standard for ISDN speech coding supports speech transmission at 64 kb/s with telephone-grade quality. It uses a sampling rate of 8000 samples per second (8 kHz) and 8 bits per sample, achieved through companding (A-law or mu-law).

1.3.1 Procedure

  • Microphone input
  • Analog-to-digital conversion (sampling at 8 kHz, producing 14‚Äì16 bits per sample)
  • Companding (compressing) before 8-bit quantization
  • Transmission to the receiver
  • De-quantization and expansion of the signal at the receiver
  • Digital-to-analog conversion including low-pass filtering

This system enables telephone conversation over a digital ISDN line at a rate of 64 kb/s.

1.4 A/D and D/A Conversion of a Signal: Quantization

1.4.1 Python Example

Assume the A/D converter has an input range of ‚àí1 V to 1 V and uses 4-bit accuracy. 

stepsize = (1.0 - (-1.0)) / (2**4)
# stepsize output: 0.12500

index = round(0.2 / stepsize)
# index output: 2

The index is encoded using 4 bits and sent to the decoder. The decoder reconstructs the voltage: 

reconstr = stepsize * index
# reconstr output: 0.25000 V

1.5 Mid-Rise and Mid-Tread Quantization

Depending on whether the quantizer has the input voltage 0 at the center of a quantization interval or on the boundary of that interval, we call the quantizer a mid-tread or a mid-rise quantizer as illustrated in Figure 1.3 [5].

1.5.1 Python Example 

import numpy as np
q = 0.1
x = np.array([0.012, -1.234, 2.456, -3.789])

# Mid-Tread quantizer (Encoder):
index = np.round(x / q)
# De-quantization (Decoder):
reconstr = index * q

# Mid-Rise quantizer (Encoder):
index = np.floor(x / q)
# De-quantization (Decoder):
reconstr = index * q + q / 2
